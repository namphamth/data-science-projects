# Data Science Projects Portfolio

## Overview

This repository contains a collection of Data Science, Machine Learning, and Data Analytics projects developed for academic research, portfolio building, and practical learning purposes.

The projects focus on real-world datasets and analytical techniques such as:

* Exploratory Data Analysis (EDA)
* Data Visualization
* Statistical Analysis
* Machine Learning
* Clustering
* Forecasting and Prediction
* Business Intelligence
